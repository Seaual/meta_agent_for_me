# .learnings 目录

本目录用于存储 Simple Note Taker 的学习记录。

## 文件说明

| 文件 | 用途 |
|-----|------|
| `feedback.md` | 用户反馈记录 |
| `patterns.md` | 学习到的模式 |

## 使用方式

### 记录反馈

当用户对笔记质量提出反馈时，将反馈追加到 `feedback.md`：

```markdown
## 2026-03-16

用户反馈：要点可以更简洁

改进措施：要点控制在15字以内
```

### 应用学习

note-taker 在处理新文本前会读取 `patterns.md`，应用已学习到的改进。

## 初始状态

本目录为初始创建，暂无学习记录。
