# Simple Note Taker

@CONVENTIONS.md

@.claude/skills/self-improving-agent/SKILL.md

---

## 使命

将任意文本转换为结构化笔记（标题 + 要点 + 总结）。

---

## Team 成员

| Agent | 职责 |
|-------|------|
| note-taker | 文本解析、结构提取、笔记生成 |

**触发词**：记笔记、整理笔记、生成笔记、summarize

---

## 上下文传递协议

本 team 为单 agent 结构，无 agent 间协作，不需要 workspace 文件传递。

自我改进相关文件：
| 文件 | 用途 |
|-----|------|
| `.learnings/feedback.md` | 用户反馈记录 |
| `.learnings/patterns.md` | 学习模式记录 |

---

## 工作流程

```
用户输入文本
    │
    ▼
┌─────────────────┐
│  note-taker     │
│  1. 解析文本    │
│  2. 提取要点    │
│  3. 生成总结    │
└─────────────────┘
    │
    ▼
Markdown 笔记输出
```

---

## 降级规则

| 情况 | 处理 |
|-----|------|
| 输入为空 | 提示用户提供文本 |
| 文本极短（<50字）| 简化结构，1-2条要点 |
| 文本过长（>5000字）| 分批处理，提示进度 |
| 无法识别主题 | 标题使用「笔记摘要」|

---

## 使用方式

直接输入文本或使用触发词：

```
帮我整理笔记：[文本内容]
```

输出格式：

```markdown
# [标题]

## 要点

1. ...
2. ...

## 总结

...

---
*生成时间：YYYY-MM-DD*
*原文长度：N 字*
```

---

## 自我改进

本 team 启用了自我改进功能。用户反馈会被记录在 `.learnings/feedback.md`，后续处理会应用改进。
