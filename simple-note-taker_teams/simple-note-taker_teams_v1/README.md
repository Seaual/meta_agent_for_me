# Simple Note Taker Team v1

一个简洁的笔记生成 Agent Team，将任意文本转化为结构化笔记。

## 功能

- 将非结构化文本转化为结构化笔记
- 支持会议记录、学习笔记、文章摘要等多种场景
- 输出格式：标题 + 要点列表 + 总结摘要
- 支持自我改进，根据用户反馈持续优化

## 安装

```bash
cp -r simple-note-taker_teams_v1/.claude/ /your/project/.claude/
```

## 使用方法

### 触发方式

在 Claude Code 中输入：

```
帮我记笔记：[文本内容]
```

或：

```
整理以下内容为笔记：[文本内容]
```

### 示例

**输入**：
```
帮我记录这个会议：今天讨论了新产品发布，市场部建议下月15号发布，技术部需要两周测试，最终决定月底发布。
```

**输出**：
```markdown
# 新产品发布计划会议

## 要点
- 发布日期：推迟至月底（原计划15号）
- 推迟原因：技术部需额外两周测试

## 总结
会议确定新产品发布延期至月底，主要原因是需要更多测试时间。
```

## Agent 说明

| Agent | 职责 | 工具权限 |
|-------|------|---------|
| note-taker | 将文本转化为结构化笔记 | Read |

## 自我改进

启用 self-improving 机制：
- 每次生成笔记后主动收集反馈
- 根据用户偏好调整输出格式
- 记住用户的格式偏好并在后续应用
