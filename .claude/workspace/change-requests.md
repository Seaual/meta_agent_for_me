# Literature Analyzer v1 → v2 升级需求

## 一、新增分析维度

### 1. 数据/代码可用性评估
- 检查论文是否提供公开数据集链接
- 检查是否有 GitHub 代码仓库
- 评估数据/代码的完整性、文档质量
- 输出可用性评分和建议

### 2. 可复现性评估
- 评估方法描述的完整性
- 检查实验参数是否充分
- 评估代码/数据对复现的支持程度
- 输出复现难度评级和建议

---

## 二、技术栈升级

### MCP 集成：GitHub
- 通过 `@anthropic-ai/mcp-server-github` 查询论文关联代码仓库
- 获取 star 数、最近更新、代码质量指标
- 为「数据/代码可用性评估」和「可复现性评估」提供数据支持

---

## 三、领域扩展

### v1 覆盖范围
- 仅聚焦「计算机与低空经济交叉领域」

### v2 覆盖范围
1. **纯计算机论文**：深度学习、算法优化、系统架构等
2. **纯低空经济论文**：政策研究、商业模式、空域管理等
3. **交叉领域论文**：无人机算法、eVTOL 智能调度等（原 v1 重点）
