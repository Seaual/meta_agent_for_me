# Phase 0 需求文档

## 用户背景
- 身份：博士生
- 原研究方向：数字足迹、LLM agent
- 目标方向：低空经济（路线规划、TSP问题）
- 目标：今年发表 SCI 一区论文
- 时间：上半年（紧迫）
- 基础：完全新手
- 每日学习时间：最多6小时

## 核心需求
**规划系统性学习路线**，帮助用户从零基础到具备发表 SCI 一区论文的能力。

## 关键约束
- 时间紧迫：上半年需完成选题+研究+投稿
- 跨领域转型：需要快速建立领域认知
- 交叉创新待定：先了解低空经济领域再决定是否融合现有技术

## 输出形式
- Markdown 文档，结构化呈现
- 中英文混合学习资料

## 外部工具集成
- GitHub：查找开源代码、复现实验
- 文献管理工具（Zotero 等）：整理参考文献
- 学术数据库：用户已有渠道

## 运行时配置
- Profile：minimal
- Self-improving：no
- Instincts：no

## Agent 数量预估
4-5个 agent：
1. 领域知识研究 agent（低空经济 + TSP + 路线规划）
2. 学习路径设计 agent（课程/文献/实践规划）
3. 资源搜集 agent（GitHub 代码 + 学术文献）
4. 时间规划 agent（里程碑 + 截止日期）
5. （可选）综合协调 agent

## 特殊考量
- 用户有 LLM agent 和数字足迹的技术积累，可能用于交叉创新
- SCI 一区要求较高，需要选择有价值的研究切入点
- 学习路线需要兼顾广度（领域概览）和深度（研究切入）
